  • Vendor: HMS Networks
  • Equipment: Ewon Flexy 202
  • Vulnerability: Cleartext Transmission of Sensitive Information

  • Ewon Flexy 202: All versions

3.2 VULNERABILITY OVERVIEW

3.2.1 Cleartext Transmission of Sensitive Information CWE-319
